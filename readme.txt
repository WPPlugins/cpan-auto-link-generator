== Description ==
This wordpress-plugin generate links. You type in your article Foo::Bar and the modul makes a CPAN-Link to this Modul.
Replace every mentioned CPAN module by a link to that module on CPAN.


== Installation ==

1. Upload `wp-cpan-autolinkgenerator.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Test it - the plugin will do links automatically when content shows
